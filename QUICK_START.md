# 🚀 クイックスタートガイド

## すぐに試す方法

### 1. ブラウザで直接開く

```bash
# index.html をダブルクリックして開くだけ！
```

ただし、一部の機能（画像読み込みなど）はローカルサーバーでの実行を推奨します。

---

### 2. Live Serverで開く（推奨）

#### VS Codeの場合：

1. VS Codeでプロジェクトフォルダを開く
2. 拡張機能「Live Server」をインストール
3. `index.html` を右クリック → "Open with Live Server"
4. ブラウザが自動で開きます（通常 `http://localhost:5500`）

#### Pythonがある場合：

```bash
# Python 3.x
python -m http.server 8000

# ブラウザで開く
open http://localhost:8000
```

#### Node.jsがある場合：

```bash
# http-serverをグローバルインストール
npm install -g http-server

# サーバー起動
http-server

# ブラウザで開く
open http://localhost:8080
```

## 💡 使い方のヒント

### カルーセルの操作

- **マウス**: ドラッグして左右にスワイプ
- **キーボード**: `←` `→` 矢印キー
- **タッチデバイス**: 指で左右スワイプ

### アルバムを開く

1. カバーをクリック
2. 本が開くアニメーションを楽しむ
3. 縦スクロールで写真を閲覧

### 戻る方法

- `ESC` キーを押す
- 左上の「← Gallery」ボタンをクリック

## 🐛 問題が起きた場合

### 画像が表示されない

→ ローカルサーバーで実行してください（上記参照）

### カルーセルが動かない

→ インターネット接続を確認（Swiper.js読み込みのため）

### アニメーションがカクつく

→ ブラウザのハードウェアアクセラレーションを有効化
